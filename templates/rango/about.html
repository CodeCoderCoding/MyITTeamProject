About

MyITTeam is the web site that provides the tour information.
The user can get the most liked cities in left area like London, Manchester, Edinburgh, Glasgow, Bath and York.
In the top, there are three buttons including home, about and logout.
In the right area, there are my views spot like Big Ben. This page can be available for all the users.
If the user wants to customize their own tour page, they need to sign up. After the user login, they will see my page.
In this page, the user can see my added cities and the user can add the cities and delete the cities.
In the meantime, the user can add the spot.
In addition, the user can click the about button to enter the about page which can learn this site how to use.
