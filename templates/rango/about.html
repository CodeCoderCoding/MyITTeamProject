About




MyITTeam is the web site that provides the British tour information. The user will enter the about page that can learn how to use this web site. The user can get popular cities in left area like London, Manchester, Glasgow and get corresponding spots at the right area in home page. In the top, there are three buttons including home, about and logout. If the user wants to customize their own tour page, they need to sign up and login. After the user login, they will see my page. In my page, the user can see my added cities and sceneries and the user can add the cities.
